# Obstapulo (Jogo em Python com Pygame)

Um jogo simples feito em Python usando Pygame, onde o jogador deve desviar de obstáculos pulando.

## Sobre o jogo
Você controla um personagem (quadrado preto) que precisa pular os obstáculos que vêm pela tela.  
O objetivo é sobreviver o maior tempo possível.

## 🎮Controles
- **ESPAÇO** → pular

## Funcionalidades
- Sistema de pulo com gravidade  
- Obstáculos aleatórios  
- Aumento de dificuldade (velocidade aumenta)  
- Detecção de colisão (game over)  
- Contador de tempo

## Como funciona
- O jogador fica fixo no eixo X e pode pular  
- Obstáculos se movem da direita para a esquerda  
- A velocidade aumenta conforme você desvia  
- O jogo termina ao colidir com um obstáculo
